## Описание проекта

Данная лабораторная работа посвящена изучению современной технологии CSS-верстки — **Flexbox**.  
В ходе работы были изучены основные свойства flex-контейнеров и flex-элементов, а также созданы практические примеры использования.

## Цель работы

Освоить адаптивную верстку с использованием Flexbox для создания гибких и отзывчивых макетов.

## Используемые технологии

- HTML5
- CSS3 (Flexbox)
- Git / GitHub
- Visual Studio Code
- Live Server
