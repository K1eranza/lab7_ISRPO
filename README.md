# Лабораторная работа №7
Целью данной лабораторной работы является изучение FlexBox и применение её для создания веб-страниц.
Освоить современные технологии CSS-разметки — Flexbox.
### Что такое FlexBox?
Flexbox — одномерная система компоновки для размещения элементов в строку или столбец.
### Адаптивная верстка 
Адаптивная верстка — это подход к созданию веб-страниц, при котором дизайн и контент автоматически подстраиваются под размер экрана устройства (компьютер, планшет, смартфон).
